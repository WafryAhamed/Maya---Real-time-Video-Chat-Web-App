# Maya - Real-Time Video Collaboration App

A modern web application for real-time video communication with integrated chat, reactions, polls, and AI assistance.

## Features

- 📹 Real-time video collaboration with WebRTC
- 💬 Live chat with typing indicators
- 🎭 Meeting reactions and emojis
- 📊 Quick polls for participants
- 🤖 AI-powered assistance
- 👥 Participant management
- 🎤 Audio/video controls
- 📱 Responsive design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```
